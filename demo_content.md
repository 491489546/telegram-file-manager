## 模板库

1. 问题-解决方案-优势-行动号召
2. 故事-冲突-转折-结局-价值
3. 痛点-利益-证据-紧迫性

### 社交媒体帖子模板
1. 问题引出 + 解决方案 + 结果
2. 成功案例 + 方法论 + 邀请参与
3. 行业洞察 + 个人观点 + 互动提问

2. 问题解决 + 产品介绍 + 优惠信息
3. 感谢 + 更新 + 继续支持邀请
